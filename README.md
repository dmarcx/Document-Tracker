# 📋 מעקב מסמכים — Document Tracker

> אפליקציית ווב פשוטה למעקב אחר תוקף מסמכים אישיים, עם התראות צבע לפי דחיפות.

---

## ✨ תכונות

| תכונה | תיאור |
|-------|-------|
| ➕ הוספת מסמך | שם, סוג, ותאריך פקיעה |
| ✏️ עריכה | עריכת כל שדה של מסמך קיים |
| 🎨 קידוד צבעים | 🔴 אדום עד 30 יום · 🟠 כתום עד 60 יום · 🟢 ירוק מעבר לכך |
| ⚙️ סוגים מותאמים | הוסף/מחק סוגי מסמכים משלך + emoji |
| ✚ הוסף סוג inline | בחר "הוסף סוג..." ישירות מה-dropdown |
| 💾 שמירה מקומית | כל הנתונים ב-`localStorage` — לא נדרש שרת |
| 📊 סטטיסטיקות | ספירת מסמכים לפי סטטוס בזמן אמת |

---

## 🚀 הרצה

```bash
# אין צורך בהתקנה — פשוט פתח את הקובץ בדפדפן:
open index.html
```

זה הכל. האפליקציה פועלת לחלוטין בדפדפן, ללא שרת, ללא build.
